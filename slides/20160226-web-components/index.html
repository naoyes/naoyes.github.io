Web Components とは

Web Components are a set of standards currently being produced by Google engineers as a W3C specification that allow for the creation of reusable widgets or components in web documents and web applications. The intention behind them is to bring component-based software engineering to the World Wide Web. The components model allows for encapsulation and interoperability of individual HTML elements.

Web Components はGoogleのエンジニアによって作成されているW3C標準仕様です。これはWeb上のドキュメントやアプリケーションで再利用可能なウィジェットやコンポーネントを作成することを可能としています。これによりWorld Wide Webにコンポーネントベースの開発をもたらすことを目的としています。コンポーネントモデルは独立したHTML要素のカプセル化と相互運用性の向上をもたらします。

Templates

    function hello() {
        var template = document.querySelector('#hello_message');
        var clone = document.importNode(template.content, true);
        document.body.appendChild(clone);
    }
</script>

HTML Imports

別ファイルとしたHTMLを取得できる

HTML Imports

// parent.html
<head>
    <meta charset="utf-8">
    <title>HTML Imports Parent</title>
    <link rel="import" href="import_common.html">
</head>
<body>
<h1>HTML Imports</h1>
<div id="content">content</div>
</body>
<script>
    var link = document.querySelector('link[rel="import"]');
    var common = link.import;

    var header = common.querySelector('#header');
    var footer = common.querySelector('#footer');
    var content = document.querySelector('#content');

    content.insertBefore(header.cloneNode(true), content.firstChild);
    document.body.appendChild(footer.cloneNode(true));
</script>
// import_common.html
<div id="header">this is header</div>
<div id="footer">this is footer</div>
